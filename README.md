# 💫 About Me

문제의 원인을 끝까지 파고들어 해결하는 프론트엔드 개발자입니다.

바이브 코딩을 하면서 하네스 구조를 끊임없이 디벨롭 중 입니다.

읽기 쉽고 확장하기 쉬운 코드를 지향하며, 팀원들이 함께 이해하고 발전시킬 수 있는 구조를 고민합니다.

현재는 프론트엔드 아키텍처를 개선하고, 더 나은 사용자 경험을 제공하는 제품을 만드는 데 관심을 가지고 학습하고 있습니다.
